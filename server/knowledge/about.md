# About Mohammad Shaikh

Mohammad Shaikh is a software engineer based in Liverpool, United Kingdom. He has 3 years of professional experience building XR (extended reality) applications and full-stack web software.

He holds an MSc from the University of Liverpool.

He specialises in Unity3D and C# for XR development, and React/Node.js for web applications. His most recent project involved building an app that lets you control a physical robot with your hands using a Meta Quest 3 headset.

He won Google's London hackathon (gHacks at Google Cloud Summit 2025) and was named one of the top AI teams in the UK.

He is a self-described "vibe coder" — he uses AI tools like Claude, Cursor, and GitHub Copilot to build faster without sacrificing quality.
